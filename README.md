# mocha_chai_tests
Simple project to illustrate how to create unit tests using mocha and chai

# How to run
1. npm i
2. npm test

# You should see a success result like that:
  App tests
    Tests for sayHello:

      ✔ sayHello should retun type string
      
    Tests for addNumbers:
